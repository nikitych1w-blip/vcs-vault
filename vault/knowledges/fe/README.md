# База знаний Frontend

## Обзор

В этой директории содержатся документация и технические спецификации для frontend-части проекта.

## Структура

- **stack.md** — Текущий frontend-стек (React 18 + TypeScript + Rsbuild)
- **legacy.md** — Легаси-стек UI (Go Templates + SSR)
- **architecture.md** — Архитектура проекта по FSD
- **styleguide.md** — React Style Guide (VCS-8306)
- **development-checklist.md** — Чеклист разработки фронта (VCS-10805)
- **wiremock.md** — WireMock Guide (VCS-8337)
- **codegen.md** — Работа с кодогенерацией (OpenAPI + Orval, VCS-8308)
- **gitea-react-adapter.md** — Gitea-React-Adapter и соответствие роутов
- **webpage-creation.md** — Как создать новую страницу на React
- **ui-system.md** — Справочник компонентов Sber Design System (@sds-eng/base)
- **tools.md** — Справочник инструментов и библиотек

## Как обновлять

При изменении технического стека обновляйте соответствующие markdown-файлы в этой директории.
