# Справочник инструментов и библиотек

## UI-фреймворки

### Fomantic UI
- **Сайт:** https://fomantic-ui.com
- **Описание:** Основной UI-кит для легаси-экранов
- **Компоненты:** Кнопки, формы, модальные окна, выпадающие списки и др.

### @sds-eng/base
- **Сайт:** https://94.139.254.218/sds-eng/1.6.0/
- **Описание:** Корпоративный UI-кит от SDS
- **Содержимое:** Готовые компоненты, токены, темы, утилиты для единообразного дизайна

## Редакторы кода

### Monaco Editor
- **Сайт:** https://microsoft.github.io/monaco-editor/
- **Описание:** Продвинутый редактор кода — ядро VS Code
- **Использование:** Подсветка синтаксиса, автодополнение, редактирование файлов

## Таблицы данных

### use Vue3 Easy Data Table
- **Сайт:** https://github.com/laraland/use-vue3-easy-data-table
- **Описание:** Готовый компонент таблицы для Vue 3
