# Легаси-стек UI

## Обзор

Описание легаси-UI — всё ещё присутствует в проекте. Архитектура зеркально повторяет Gitea 2024 года: Go-шаблоны рендерят HTML на сервере, интерактивность добавляется смесью Vue, jQuery и чистого JS.

**Ключевые характеристики:**
- Единого стандарта нет — логика зависит от экрана
- Глобальные данные в `window.config`, доступны любому скрипту
- Глобальные стили (без модулей) — правки требуют осторожности

## Технологии

| Технология | Описание |
|------------|-------------|
| Go Templates | Серверный рендеринг HTML ([pkg.go.dev/html/template](https://pkg.go.dev/html/template)) |
| Vue.js | Реактивные компоненты интерфейса |
| jQuery | Манипуляции с DOM в легаси-скриптах |
| Fomantic UI | Основной UI-кит (кнопки, формы, модальные окна) |
| Monaco Editor | Редактор кода и подсветка синтаксиса в файлах |
| Dropzone | Drag-and-drop загрузка файлов |
| Clipboard.js | Копирование текста в буфер обмена |
| Day.js | Работа с датами |
| Webpack | Сборщик модулей |
| KaTeX | Рендеринг математических формул |
| Mermaid | Генерация диаграмм из текста |
| Tippy.js | Всплывающие подсказки |
| Tribute.js | Автокомплит и упоминания |
| use Vue3 Easy Data Table | Готовый компонент таблицы для Vue 3 |

## Статус миграции

Планируется полная перепись этих экранов, но текущее понимание механики необходимо для:
- Поддержки существующего функционала
- Предотвращения разрушения существующих связей

## Руководство по миграции

При рефакторинге легаси-экранов:
1. Документируй существующее поведение до изменений
2. Тестируй по существующим данным/потокам
3. Сохраняй паттерны доступа к `window.config` до завершения миграции
4. Избегай глобальных CSS-эффектов
