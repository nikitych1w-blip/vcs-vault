# Продукт: SourceControl

**SourceControl** — корпоративная платформа управления исходным кодом (VCS), построенная на базе Gitea.

## Интерфейсы

| Интерфейс | Тег | Описание |
|-----------|-----|----------|
| Web UI | `#UI` `#FE` | Браузерный интерфейс управления проектами и репозиториями |
| REST API | `#api` | Несколько версий: v1, v2, v3, web1, web2 (Новый FE) |
| CLI | `#CLI` | Командный интерфейс |

## Версии API

| Версия | Тег | Приоритет |
|--------|-----|-----------|
| REST v1 | `#v1` `#ПАО` | Приоритетное покрытие |
| REST v2 | `#v2` `#ПАО` | Приоритетное покрытие |
| REST v3 | `#v3` | — |
| Frontend web1 | `#web1` | — |
| Frontend web2 | `#web2` `#ПАО` `#FE` | Приоритетное покрытие, новый FE |

## Интеграции

| Система | Тег | Роль |
|---------|-----|------|
| Git | `#git` `#ПАО` | Push/pull операции |
| Kafka | `#kafka` `#ПАО` | События системы |
| Jenkins / Pipeliner | `#pipeline` | CI/CD |
| SonarQube | `#sonar` | Качество кода |
| LDAP | `#ldap` | Аутентификация |

## Основные сущности

| Сущность | Тег | Описание |
|----------|-----|----------|
| Проект | `#project` | Организационная единица, содержит репозитории |
| Репозиторий | `#repo` | Git-хранилище внутри проекта |
| Ветка | `#branch` | Именованная ссылка на цепочку коммитов |
| Коммит | `#commit` | Снимок состояния с метаданными |
| Тег | `#tags` | Именованная метка на коммите |
| Релиз | — | Версия с тегом и артефактами |
| Запрос на слияние | `#merge` `#request` | Запрос на слияние веток |
| Метка | `#marker` | Пользовательская категоризация |
| Форк | `#fork` | Независимая копия репозитория |

## Аутентификация

- `#ldap` — LDAP
- `#access-token` — Access Token для API
- `#jwt` / `#jwt-token` — JWT Token для API
- Требования к учётным записям (парольная политика и пр.)

## Важно: deprecated

Узлы, помеченные `(удалить раздел)` в test-model (например, «Команды»), считаются deprecated:
- SA не анализирует их
- QA не пишет на них TC и деактивирует существующие
- Разработчик не реализует новую функциональность
