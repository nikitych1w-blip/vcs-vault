# FAQ — 04-vcs-arch

**Что это?**
Внутренний репозиторий с архитектурными материалами платформы SourceControl: C4-диаграммы, ADR, PlantUML-схемы.

**Где лежит?**
`sources/vcs-arch/`

**Как обновить?**
```bash
make pull SOURCES=04-vcs-arch
```

**Требования**
SSH-доступ к внутреннему Gitea. Источник помечен `optional: true`.

**Что внутри?**
Диаграммы контекста/контейнеров/компонентов, Architecture Decision Records, схемы деплоя.
