# Инструкция по настройке npm

## Описание

Файл `.npmrc` используется для конфигурации поведения npm. В проекте присутствует шаблон `.npmrc.example`, который можно использовать как основу для создания собственного конфигурационного файла.

## Шаги настройки

1. **Создайте копию шаблона**

   ```bash
   cp .npmrc.example .npmrc
   ```

2. **Откройте файл `.npmrc` в текстовом редакторе**

3. **Настройте параметры**
   - Убедитесь, что все необходимые переменные окружения и настройки указаны.
   - Замените base64_string, token на свои.

## Где может находиться файл `.npmrc`

### 1. **В корне проекта** (рекомендуемое место)

```
project-root/
├── package.json
├── .npmrc          # ← Файл конфигурации npm
└── .npmrc.example  # ← Шаблон конфигурации
```

### 2. **В домашней директории пользователя**

```
~/.npmrc            # ← Глобальная конфигурация для всех проектов
```

### 3. **В каталоге npm cache**

```
~/.npm/_npx/        # ← Кэш npm (не рекомендуется для ручной настройки)
```

### 4. **В директории конкретного модуля**

```
node_modules/package-name/.npmrc
```

## Приоритеты загрузки конфигурационных файлов

npm загружает конфигурацию в следующем порядке (от высшего к низшему приоритету):

1. **Командная строка** (`npm install --registry=https://registry.npmjs.org/`)
2. **Локальный `.npmrc`** (в текущей директории или выше)
3. **Глобальный `.npmrc`** (`~/.npmrc`)
4. **Значения по умолчанию** (встроенные значения npm)
