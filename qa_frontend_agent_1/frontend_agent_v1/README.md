# Frontend Agent v1

Агент для автоматизации ручного тестирования frontend-сценариев (UI/FE) через систему управления тестированием (TMS).

---

## Структура проекта

```
frontend_agent_v1/
├── promt_v1.txt              # Основной промт с инструкциями
├── shablon.md                # Шаблон для тест-кейсов
├── test_results_shablon.json # Шаблон отчёта о тестировании
├── config.md                 # Конфигурация (учётные данные)
├── generate_vcs_data.py      # Скрипт генерации тестовых данных
├── import_tests_to_tms_full.py # Скрипт импорта в TMS
└── README.md                 # Этот файл
```

---

## Требования

- Python 3.x (для скриптов импорта и генерации данных)
- Доступ к TMS: `https://portal.works.prod.sbt`
- Доступ к VCS Frontend: `https://sc-vp.ift.pd04.pvw.sbt/web/v2/`
- Учётные данные в файле `config.md`

---

## Использование

### Быстрый старт

1. **Подготовка окружения:**
   ```bash
   # Убедитесь, что config.md содержит актуальные учётные данные
   python generate_vcs_data.py --base-url "https://sc-vp.ift.pd04.pvw.sbt/api/v3" \
       --tenant "2afda79a-ae63-47f8-b0d3-1f67ea07b92c" \
       --project "agentvp"
   ```

2. **Запуск агента:**

   Откройте файл `promt_v1.txt` и скопируйте его содержимое в чат с агентом Cline.

3. **Этапы выполнения:**
   - Этап 1: Анализ UI-узлов и спецификации
   - Этап 2: Генерация тест-кейсов для UI
   - Этап 3: Анализ структуры папок в TMS
   - Этап 4: Импорт тест-кейсов в TMS
   - Этап 5: Ручное тестирование и формирование отчёта

---

## Формат тест-кейсов

Тест-кейсы создаются по шаблону из файла `shablon.md`:

```markdown
## TC-001: [Название теста]

**Id:** TC-001
**Название:** [Название]
**Приоритет:** critical/major/minor
**Уровень теста:** ui
**Теги:** ui, [функционал], web2
**Шаги:**

**Шаг 1:** [Описание шага]
**Тестовые данные:** [Данные]
**Ожидаемый результат:** [Ожидаемое поведение]
```

---

## Формат отчёта

Отчёты создаются по шаблону из файла `test_results_shablon.json`:

```json
{
  "summary": {
    "total": 10,
    "passed": 8,
    "failed": 2,
    "success_rate": "80.00%"
  },
  "results": [...]
}
```
